# Protein-In

> Calories are so last year.

Protein-first nutrition database. Search any food and find out how much protein it contains.

## Stack

- Laravel 11, PHP 8.3
- MySQL 8
- Vite + vanilla JS

## Local development

```
make dev
```

Runs at [http://localhost:8002](http://localhost:8002).

## Other commands

| Command | Description |
|---------|-------------|
| `make start` | Start containers in background |
| `make stop` | Stop containers |
| `make logs` | Tail logs |
| `make shell` | Shell into app container |
| `make db` | MySQL CLI |
| `make rebuild` | Full rebuild (no cache) |
