# Traitor Sites

A collection of small web projects.

## Sites

### Facemaker Pottery (`facemaker/`)

A static HTML e-commerce site for a handcrafted ceramic pottery shop. Features a shop with basket functionality (session storage), product listings, checkout flow, and an about page. No build step — plain HTML, CSS, and vanilla JS.

### Top Toast (`toptoast.com/`) — [README](toptoast.com/CLAUDE.md)

A PHP recipe site for creative toast recipes. Includes a homepage, recipe listings, individual recipe pages, and an about page. Built with plain PHP includes and a flat file structure — no framework or database required.
